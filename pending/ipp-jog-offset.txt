Per the PWG February 2017 face-to-face meeting, the IPP WG consensus was to
deprecate the 'jog-offset' finishing process since it is better supported by
the 'mailbox' and 'stacker' values of the "output-bin" Job Template attribute.

The following registration template deprecates the 'jog-offset' value of the
"finishings" Job Template attribute and "finishing-template" member attribute.


Attributes (attribute syntax)
  Keyword Attribute Value		Reference
  -----------------------		---------
finishing-template (type2 keyword | name(MAX))	[PWG5100.3]
  jog-offset(deprecated)               	[IPP-JOG-OFFSET]


Attributes (attribute syntax)
  Enum Value	Enum Symbolic Name	Reference
  ----------	------------------	---------
finishings (type2 enum)			[RFC8011]
  14            jog-offset(deprecated)  [IPP-JOG-OFFSET]
